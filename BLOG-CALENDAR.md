## Content Guidelines

### Post Structure
1. **Hook** (1-2 sentences): What problem does this solve?
2. **Context** (1 paragraph): Why does this matter?
3. **Solution** (bulk): Code examples, explanations, diagrams
4. **Summary**: Key takeaways as bullet points
5. **CTA**: Link to docs, GitHub, or related post

### SEO Checklist Per Post
- [ ] Title includes primary keyword
- [ ] Meta description (150-160 chars) with keyword
- [ ] H2/H3 hierarchy with keyword variations
- [ ] At least 1 code example
- [ ] 2-3 internal links to docs pages
- [ ] At least 1 external reference
- [ ] 800+ words minimum

### AI Citation Optimization
- Include quotable statistics/benchmarks
- Use comparison tables (AI loves structured data)
- Write clear "what is X" definitions
- Answer "how to" questions directly in the first paragraph
